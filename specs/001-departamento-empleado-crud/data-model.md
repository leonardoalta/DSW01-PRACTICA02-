# Data Model - CRUD de Departamentos relacionados con Empleados

## Entity: Departamento

### Fields
- `id` (long, required, unique, PK, autogenerated)
- `nombre` (string, required, unique, maxLength: 100)
- `descripcion` (string, optional, maxLength: 255)
- `createdAt` (datetime, required)
- `updatedAt` (datetime, required)

### Validation Rules
- `nombre` no puede ser nulo ni vacío.
- `nombre` debe ser único a nivel global.
- `nombre` no puede exceder 100 caracteres.
- `descripcion`, cuando se envía, no puede exceder 255 caracteres.

## Entity: Empleado (impactado)

### Existing identity (preserved)
- `clave` (string, required, unique, PK canónica de autenticación)

### New relation field
- `departamentoId` (long, optional, FK -> `Departamento.id`)
  - Regla: un empleado puede pertenecer a 0..1 departamento vigente.

### Validation Rules
- No se puede asociar un empleado a un `departamentoId` inexistente.
- La reasignación reemplaza la referencia previa.

## Relationships
- `Departamento` 1 ---- * `Empleado`
  - Un departamento tiene 0..N empleados.
  - Un empleado tiene 0..1 departamento.

## State/Behavior Rules
- Eliminar departamento con empleados asociados => conflicto de estado.
- Eliminar departamento sin empleados asociados => permitido.
- Asignar empleado a departamento => actualiza `departamentoId` del empleado.
- Reasignar empleado => sobreescribe `departamentoId` previo.

## DTOs

### DepartamentoCreateRequest
- `nombre` (required, maxLength 100)
- `descripcion` (optional, maxLength 255)

### DepartamentoUpdateRequest
- `nombre` (required, maxLength 100)
- `descripcion` (optional, maxLength 255)

### DepartamentoResponse
- `id` (required)
- `nombre` (required)
- `descripcion` (optional)

### DepartamentoDetalleResponse
- `id` (required)
- `nombre` (required)
- `descripcion` (optional)
- `cantidadEmpleados` (required, >= 0)
- `empleados` (required, array de resumen de empleado)

### AsignacionDepartamentoResponse
- `claveEmpleado` (required)
- `departamentoId` (required)
- `departamentoNombre` (required)
